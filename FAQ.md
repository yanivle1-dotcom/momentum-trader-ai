## 🎯 מסחר וסט-אפים

### ש: האם המערכת אוטומטית סוחרת בשבילי?
**ת:** **לא!** המערכת רק מנתחת ומציעה. אתה מחליט אם להיכנס או לא.

זה כלי עזר, לא רובוט מסחר.

### ש: מה ההבדל בין הסט-אפים?
**ת:**
- **Gap & Go**: גאפ חזק + פריצה
- **Red to Green**: מאדום לירוק = שינוי סנטימנט
- **First Green Day**: יום ירוק אחרי ירידות
- **Micro Pullback**: כניסה במגמה קיימת
- **Bull Flag**: קונסולידציה לפני המשך עלייה

ראה [ROSS_CAMERON_GUIDE.md](ROSS_CAMERON_GUIDE.md) להרחבה!

### ש: מה זה R:R?
**ת:** **Risk:Reward Ratio** - יחס סיכון/תגמול.

דוגמה:
- סיכון: $0.50
- תגמול: $1.50
- R:R = 1:3

Ross ממליץ מינימום 1:2.

### ש: למה אני צריך Stop Loss?
**ת:** בלי Stop = אסון מובטח!

Ross: "Cut losses quickly, let winners run."

Stop Loss:
- מגן על הון
- שומר על משמעת
- מונע הפסדים ענקיים

**תמיד הגדר Stop לפני הכניסה!**

## 🛠️ בעיות טכניות

### ש: "No module named flask" - מה עושים?
**ת:**
```bash
pip install -r requirements.txt
```

אם עדיין לא עובד:
```bash
python3 -m pip install flask
```

### ש: "Agent not configured" - מה הבעיה?
**ת:** בדוק:
1. קיים קובץ `.env`?
2. יש בו מפתח API תקין?
3. המפתח לא פג תוקף?

נסה:
```bash
cat .env  # הצג את התוכן
```

### ש: שגיאה: "Port 5000 already in use"
**ת:** כבר יש משהו שרץ על פורט 5000.

**פתרון 1**: שנה פורט ב-`.env`:
```bash
FLASK_PORT=5001
```

**פתרון 2**: סגור את התהליך הישן:
```bash
lsof -ti:5000 | xargs kill -9
```

### ש: הגרפים לא נטענים
**ת:** בדוק:
1. חיבור אינטרנט פעיל?
2. פתח Console בדפדפן (F12) - יש שגיאות?
3. נסה דפדפן אחר

Plotly נטען מ-CDN, צריך אינטרנט.

### ש: "SSL Certificate Error"
**ת:** בעיית אישור אבטחה.

**פתרון זמני** (לפיתוח בלבד!):
```python
# בתחילת app.py
import ssl
ssl._create_default_https_context = ssl._create_unverified_context
```

**אזהרה**: רק לפיתוח מקומי!

```json
    "NVDA",

  ]
```

שמור, רענן דפדפן - זהו!

### ש: איך אני משנה את קריטריוני הסריקה?
**ת:** באותו קובץ:

```json
{
  "momentum_criteria": {
    "min_rvol": 1.5,         // הורד מ-2.0
    "min_gap_percent": 2.0,  // הורד מ-3.0
    "min_volume": 50000      // הורד מ-100000
  }
}
```

**שים לב**: קריטריונים רכים = יותר תוצאות אבל פחות איכותיות.

### ש: אפשר להוסיף עוד אינדיקטורים?
**ת:** כן! ערוך `src/data/market_data.py`:

```python
df['SMA_50'] = ta.sma(df['Close'], length=50)
df['MACD'] = ta.macd(df['Close'])
```

pandas-ta תומך בעשרות אינדיקטורים!
